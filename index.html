Entendendo as Métricas (Glossário)

Tempo de Resposta (ms)
Mede quanto tempo (em milissegundos) o sistema leva para responder a uma requisição. Valores mais baixos indicam um sistema mais rápido e saudável.
Serviços
Um sistema de software é composto por várias partes que trabalham juntas, aqui chamadas de "serviços" (ex: API, Banco de Dados). Analisar o desempenho de cada um ajuda a isolar problemas.
Coeficiente de Variação (CV)
É uma medida de dispersão que indica a consistência dos dados. No nosso contexto, um CV baixo para os tempos de resposta significa que o sistema se comporta de maneira previsível e estável. Um CV alto sugere instabilidade.
Histograma vs. Gráfico de Barras
Ambos usam barras, mas o Histograma agrupa dados numéricos contínuos (como o tempo de resposta) em "classes" ou "faixas" para mostrar onde os valores se concentram. O Gráfico de Barras compara contagens de categorias distintas e separadas (como os diferentes serviços ou códigos de status).
